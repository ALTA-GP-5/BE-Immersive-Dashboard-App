# Immersive Dashboard App - Documentation.

## Required Stuff:

- Docker
- Docker Compose

## Required Step:

Copy `.env-example` and rename to `.env`

## Usage:

Command to run dev environment :
```bash
go mod tidy
```

Command to run dev environment :
```bash
./run-dev.sh
```

Command to start testing :
```bash
./test.sh
```
